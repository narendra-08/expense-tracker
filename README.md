# Expense Tracker (Full Stack Project)

## Project Description
Expense Tracker is a full stack web application that helps users manage their income and expenses.
Users can add, delete, filter and search transactions.
The project also shows expense analysis using charts.

The application is built using a separate frontend and backend architecture.

---

## Features
- User Login and Signup
- Add Income and Expense Transactions
- Delete Transactions
- Filter Transactions by Month and Type
- Search Transactions by Category or Note
- Real-time Balance Calculation
- Expense Analysis using Charts
- Export Transactions as CSV

---

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Chart.js
- Vite

### Backend
- Node.js
- Express.js

## How to Run the Project Locally

### Backend
```bash
cd backend
npm install
node server.js
```
### Frontend
```bash
cd frontend
npm install
npm run dev
```
